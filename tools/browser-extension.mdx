The Summate browser extension lets you save articles, summarize pages, and add content to your digests without leaving your browser.

## Installation

### Chrome Web Store

1. Visit [Chrome Web Store](https://chrome.google.com/webstore/summate)
2. Click "Add to Chrome"
3. Confirm permissions
4. Pin to toolbar for easy access

### Manual Installation

For beta testing:
1. Download extension package
2. Chrome → Extensions → Developer mode
3. Load unpacked extension
4. Select extension folder

## Features

### Quick Save

Save any webpage to read later:
- Click extension icon
- Select "Save for Later"
- Choose digest (optional)
- Article saved with summary

**Keyboard Shortcut:** `Cmd/Ctrl + Shift + S`

### Instant Summarize

Get AI summary without saving:
- Click extension icon
- Select "Summarize"
- View key points instantly
- Copy or share summary

**Keyboard Shortcut:** `Cmd/Ctrl + Shift + Y`

[Screenshot placeholder: Summary popup]

### Add to Digest

Add current page to specific digest:
- Click extension icon
- Select "Add to Digest"
- Choose target digest
- Set priority (optional)
- Appears in next digest

### Smart Detection

Extension automatically detects:
- Article content
- Video pages
- Newsletter signups
- RSS feeds
- Paywall status

## Keyboard Shortcuts

### Global Shortcuts

- `Cmd/Ctrl + Shift + S` - Save current page
- `Cmd/Ctrl + Shift + Y` - Summarize page
- `Cmd/Ctrl + Shift + D` - Open digest selector

### Reading Mode

- `R` - Toggle reading mode
- `D` - Toggle dark mode
- `+/-` - Adjust font size
- `Esc` - Exit reading mode

### Customization

Change shortcuts:
1. Chrome → Extensions → Shortcuts
2. Find Summate
3. Set custom combinations
4. Save changes

## Troubleshooting

### Extension Not Working

1. Check you're signed in
2. Refresh the page
3. Disable conflicting extensions
4. Clear extension cache
5. Reinstall if needed

### Can't Save Page

Possible issues:
- Paywall blocking content
- Site blocking extensions
- JavaScript required
- Try reader mode first

### Sync Issues

If saves don't appear:
- Check internet connection
- Verify account status
- Manual sync button
- Check credit balance

## Privacy & Security

### Permissions

Extension requires:
- **Active tab** - Read current page
- **Storage** - Save preferences
- **Context menus** - Right-click options

### Data Handling

- Content processed on Summate servers
- No browsing history tracked
- Saves only what you select
- Delete data anytime

### Security

- HTTPS only
- Token-based auth
- No password storage
- Regular security audits
