## API Access

### Webhooks

Trigger actions on:
- Digest sent
- Item saved

## Setting Up Integrations

### General Process

1. Navigate to Settings ‚Üí Integrations
2. Choose service to connect
3. Authorize access
4. Configure preferences
5. Test connection

### Security

All integrations use:
- OAuth 2.0
- Minimal permissions
- Revokable access
- Encrypted storage

### Troubleshooting

**Connection Failed?**
- Check permissions
- Re-authorize
- Clear cache
- Contact support

**Sync Issues?**
- Manual sync button
- Check rate limits
- Verify credentials
- Review logs
