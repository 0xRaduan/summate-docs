Blocks are content sections within your digest. Each block type connects to different sources and presents content optimized for that medium.

## ðŸ“§ Newsletter Blocks

Pull in content from email newsletters and subscriptions.

### How It Works

1. **Forward newsletters** to your AI inbox: `username@inbox.summate.io`
2. **Auto-discovery** - Summate identifies newsletters in your inbox
3. **Select sources** - Choose which newsletters to include
4. **Smart filtering** - Automatically approve known publishers

### Features

- **Sender management** - Approve or block specific senders
- **Content detection** - Identifies newsletter vs transactional emails
- **Archive access** - Include past newsletters in summaries

[Screenshot placeholder: Newsletter block configuration]

### Best Practices

- Forward existing subscriptions vs resubscribing
- Use filters in your email client for bulk forwarding
- Review auto-approved senders monthly

## ðŸ“º YouTube Blocks

Summarize videos from YouTube channels you follow.

### Setup Options

**Quick Add**
- Paste channel URLs
- Search by channel name
- Import from YouTube account

**Bulk Import**
1. Connect YouTube account
2. Select subscribed channels
3. Choose import all or specific channels

[Screenshot placeholder: YouTube block with channel list]

### Configuration

- **Video count** - How many recent videos to include
- **Time range** - Videos from last X days
- **Duration filter** - Skip videos under/over certain length
- **Type filter** - Exclude shorts, premieres, or live streams

### Summary Features

- **Timestamps** - Jump to specific moments
- **Key takeaways** - Main points with timecodes
- **Visual markers** - Screenshots at key moments

## ðŸ“° RSS Blocks

Aggregate content from blogs, news sites, and any RSS/Atom feeds.

### Adding Sources

**Automatic Detection**
- Enter website URL
- Summate finds RSS feed
- Preview available feeds

**Direct Feed URL**
- Paste RSS/Atom feed URL
- Useful for hidden feeds
- Support for authenticated feeds

**Popular Sources**
- Tech: TechCrunch, Ars Technica, The Verge
- News: Reuters, AP, BBC
- Blogs: Medium publications, Substack, Ghost

### Advanced Options

- **Full text extraction** - Fetch complete articles
- **Image inclusion** - Show article images
- **Category filtering** - Include specific topics only
- **Author filtering** - Follow specific writers

## Combining Block Types

### Effective Combinations

**Comprehensive Coverage**
- Newsletter + YouTube + RSS on same topic
- Different perspectives and formats
- Avoid information gaps

**Source Hierarchy**
1. Newsletters for analysis
2. YouTube for tutorials
3. RSS for breaking news

**Time-Based Mixing**
- Morning: Newsletter highlights
- Lunch: Quick RSS updates
- Evening: YouTube summaries

## Block Management

### Organizing Blocks

- **Drag to reorder** - Most important first
- **Group by type** - All newsletters together
- **Mix by topic** - Varied sources per subject

### Block Settings

Each block has:
- **Title** - Custom name for the block
- **Item limit** - Max items to include
- **Sort order** - Newest first or by relevance
- **Filters** - Keywords, authors, categories

### Performance Tips

- Limit blocks to 5-7 per digest
- Balance content types
- Consider reading time
- Test different combinations
