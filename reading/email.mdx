Get your personalized digests delivered directly to your inbox.

## Email Format

### Subject Lines

Customize how digests appear:
- Default: `Your [Digest Name] for [Date]`
- Custom: Add your own format
- Variables: `{digest}`, `{date}`, `{day}`

Examples:
- "Your Morning Brief for Monday, July 21"
- "AI Weekly: July 21 Edition"
- "☀️ Monday Morning Digest"

### Email Structure

Each digest email includes:

1. **Header**
   - Digest name and date
   - Quick stats (items, read time)
   - Web version link

2. **Table of Contents**
   - Jump links to sections
   - Item counts per block
   - Total reading time

3. **Content Blocks**
   - Block title and emoji
   - Individual summaries
   - Source attribution

4. **Footer**
   - Manage preferences
   - Pause digest
   - Feedback link

## Email Management

### Delivery Settings

**Sender Name**
- Default: "Summate Digest"
- Custom: "Your AI Assistant"
- Personal: "Morning Reading"

**Reply-To Address**
- No-reply default
- Your email
- Team email

### Filtering & Rules

Create email rules for:
- Auto-filing in folders
- Marking as important
- Forwarding to others
- Calendar integration

**Gmail Example:**
```
From: digest@summate.io
Subject: Morning Brief
Action: Apply label "Daily Reading"
```

### Preventing Spam

Ensure delivery:
1. Add `digest@summate.io` to contacts
2. Check spam folder initially
3. Mark as "Not Spam"
4. Create filter to never spam

## Troubleshooting

### Not Receiving Emails?

Check:
- ✓ Correct email address
- ✓ Digest is active
- ✓ Not in spam/junk
- ✓ Email filters
- ✓ Storage quota

### Formatting Issues?

Try:
- Web version link
- Different email client
- Disable email signatures
- Check security settings

### Delayed Delivery?

Possible causes:
- Email provider delays
- Large digest size
- Server maintenance
- Time zone mismatch
